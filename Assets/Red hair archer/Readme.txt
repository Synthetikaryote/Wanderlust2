The package contains:

Test scene
	
	Model lit by three lights, stringing all animations.

Animation folder
	
	animation controller with all animations assigned to the model. 

Material folder

	1 material for the model.
	1 material for shoulder pads and belts.
	1 material for the arc

Meshes folder

	1 Red hair archer model
	1 Carcaj model
	1 Arc model

Prefabs folder

	1 Red hair archer prefab
	1 Arc prefab

Texture folder

	2048 textures for models and assets and 512 textures for the arc and carcaj.
	1024 textures folder

		1024 textures for model and assets.
